# Examen Automation -- Joan Tasayco

## Installation

> 1. Instalar Java (JDK) ultima versión

> 2. Instalar un IDE para el desarrollo del script, por ejemplo Eclipse

> 3. Instalar la libreria Junit.

> 4. Instalar la libreria selenium para Java.

> 5. Instalar Maven para Java.

> 6. Listo, ya podemos crear nuestros script automation con el framework Cucumber

> 7. En el archivo pom.xml definimos las dependencias como junit, selenium,cucumber

> 8. Realizar la construccion de script automatizado
